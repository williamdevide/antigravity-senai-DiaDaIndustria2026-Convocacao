# Projeto Convocação - Dia da Indústria 2026 - SENAI

Site oficial para a convocação e orientações do Dia da Indústria 2026 no SENAI Ourinhos.

## 🚀 Tecnologias
- HTML5 / CSS3 / JavaScript Vanilla
- GitHub Actions (Auto-deploy)
- GitHub Pages

## 📂 Estrutura
- `index.html`: Página inicial de seleção.
- `1-ano.html`: Desafios para o 1º ano.
- `2-ano.html`: Desafios para o 2º ano.
- `assets/`: Imagens e logos oficiais.
- `Especificação *.pdf`: Documentação detalhada dos desafios.
